AGREEMO Project Overview

Executive Summary

AGREEMO is a smart, solar-powered hydroponic greenhouse system integrated with IoT and AI technologies, designed for lettuce cultivation at the Sunnyville Community Model Farm. It mitigates manual monitoring inefficiencies and improves crop yield through an Android-based control system and a web-based management dashboard.

Objectives

General Objective

Develop a solar-powered hydroponic greenhouse with an Android application for plant monitoring and an automatic nutrient controller tailored for urban agriculture.

Specific Objectives

  • Identify problems in existing hydroponic greenhouse operations.
  • Automate misting and cooling systems to manage internal temperature.
  • Utilize rainwater and solar energy to reduce reliance on traditional utilities.
  • Enable real-time monitoring and control of greenhouse functions via mobile app.
  • Allow farm managers to manage users and review farm data via a web platform.
  • Evaluate the system based on ISO 25010 software quality attributes.

Technology Stack

  • Microcontrollers: ESP32 for data collection and hardware control
  • Sensors: pH, TDS, humidity, temperature, and ultrasonic sensors
  • AI Module: YOLOv8 for image-based lettuce growth recognition
  • Mobile App: Android (minimum version 11) using Flutter
  • Web Dashboard: Flask-based backend with real-time analytics
  • Notifications: OneSignal for harvest alerts and sensor updates
  • Data Handling: OpenCV & NumPy for image processing, Firebase for sensor data, PostgreSQL for structured data
  • Power: Solar panels with battery storage and rainwater harvesting integration

Key Features

  • Real-time temperature and humidity monitoring with automated misting
  • AI-based lettuce maturity detection and harvest alert system
  • Ultrasonic water level detection and refill notifications
  • Manual override for nutrient dispensing and pH balancing
  • Web interface for activity logging, inventory management, and system health reports

Conclusion

AGREEMO significantly enhances efficiency and sustainability in urban hydroponic farming. By integrating IoT, renewable energy, and real-time data analytics, the system reduces manual labor, ensures optimal growing conditions, and improves crop yield. It stands as a benchmark for future smart agriculture innovations.
