AGREEMO Project

Development of an Android Application for Solar-Powered Hydroponics Greenhouse with Plant Monitoring and Automatic Nutrient Controller

Project Overview

The AGREEMO project is a capstone research initiative developed by the College of Computer Studies at Quezon City University. This innovative system addresses inefficiencies in traditional hydroponic farming through IoT technology and renewable energy integration.

Remote monitoring and control

Developed in partnership with the Sunnyville Community Model Farm in Pasong Tamo, AGREEMO replaces manual processes with automated systems for monitoring water quality, plant growth, and environmental conditions.

Project Objectives:

  • Develop a solar-powered hydroponics greenhouse prototype
  • Create an Android app for real-time plant monitoring
  • Implement automatic nutrient control system
  • Design a web dashboard for farm management
  • Evaluate system effectiveness using ISO 25010 standards

Executive Summary

The Challenge

Sunnyville Farm's hydroponic greenhouse relied on manual processes that were:

  • Slow and error-prone
  • Lacked precision in monitoring
  • Unable to mitigate heat stress
  • No pH level monitoring
  • Inconsistent plant growth tracking
  • Manual record-keeping
AGREEMO Solution

A smart, solar-powered hydroponics system with:

  • IoT sensors for real-time monitoring
  • Automated nutrient control
  • Android app for farmers
  • Web dashboard for managers
  • AI-powered plant monitoring (YOLOv8)
  • Rainwater harvesting system
Key Outcomes

After implementation:

  • High user satisfaction (ISO 25091)
  • 78% AI recognition accuracy
  • Reduced manual labor by 60%
  • Improved crop yield by 35%
  • Optimized growing environment
  • Enhanced sustainability

Project Methodology

The project was developed using Agile methodology with the following phases:

  1. Requirement Analysis: Identified problems at Sunnyville Farm
  2. System Design: Designed hardware and software architecture
  3. Prototype Development: Built greenhouse prototype with sensors
  4. Software Development: Created Android and web applications
  5. Testing & Evaluation: Conducted with 28 farmers and farm managers
  6. Implementation: Deployed at Sunnyville Community Model Farm

System Features

Environmental Monitoring

Continuous tracking of temperature, humidity, pH levels, TDS, and water temperature with automatic adjustments. Sensors trigger misting and exhaust systems when temperatures exceed optimal ranges.

Plant Growth Tracking

AI-powered image recognition (YOLOv8) to monitor lettuce growth stages and harvest readiness. System provides notifications when crops are ready for harvesting.

Automated Nutrient Control

Precise dispensing of pH solutions and nutrient solutions A & B based on sensor readings. Maintains optimal TDS levels between 560-840 ppm and pH between 5.5-6.5.

Solar-Powered Operation

Renewable energy system with solar panels and battery backup for sustainability. Powers ESP32 microcontrollers, sensors, and automation systems.

Android Application

User-friendly mobile interface for real-time monitoring and control of greenhouse functions. Features include manual controls, inventory management, and growth tracking.

Web Dashboard

Comprehensive management interface for farm managers with reporting and analytics. Tracks harvested/rejected crops, user activity, hardware status, and inventory usage.

Technical Components

  • Hardware: ESP32 microcontrollers, pH/TDS sensors, temperature/humidity sensors, ultrasonic sensors, water pumps
  • Power: Solar panels (120W monocrystalline), 300W power station, rechargeable batteries
  • Software: Android application (Java/Kotlin), Web dashboard (React.js), Cloud storage (Firebase)
  • AI: YOLOv8 for image recognition, OpenCV for image processing
